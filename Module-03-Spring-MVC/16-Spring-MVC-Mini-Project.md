# Spring MVC Mini Project

## 🎯 Project Goal

Build a Student Registration Portal using Spring MVC and Thymeleaf.

The application should:

- Display a registration form.
- Accept user input.
- Validate the input.
- Display validation errors.
- Show a success page after successful registration.

No database is used in this module.

---

## 📚 Concepts Practiced

- Controller
- Request Mapping
- GET & POST requests
- Model
- Thymeleaf
- Form Binding
- @ModelAttribute
- Bean Validation
- BindingResult

## 🚀 Workflow

GET /students/register

↓

Display Registration Form

↓

User Fills Form

↓

POST /students/register

↓

Data Binding

↓

Validation

↓

If Errors

↓

Return Form

↓

Else

↓

Success Page

## 🎯 Learning Outcome

After completing this project you should be able to build a complete Spring MVC application that accepts user input, validates it, and renders dynamic HTML pages.
